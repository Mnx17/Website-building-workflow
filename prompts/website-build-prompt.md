# Website Build Prompt

You are building a production-quality website in this repo.

## Requirements

- Create a complete responsive site, not a landing placeholder
- Use modern component structure
- Keep accessibility and performance in mind
- Use Framer Motion (`motion`) for meaningful interaction and transitions
- Keep styles maintainable
- Avoid boilerplate-looking sections

## Workflow

1. Inspect current repo structure
2. Propose final page map and component tree
3. Implement pages/components
4. Verify build with `npm run build`
5. Leave a concise implementation summary
